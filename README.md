   GitHub Desktop Tutorial

[![GitHub](https://img.shields.io/badge/GitHub-Desktop-181717?style=for-the-badge&logo=github)](https://desktop.github.com/)
[![License](https://img.shields.io/badge/License-MIT-yellow?style=for-the-badge)](LICENSE)

Repositório educacional com tutoriais e exemplos práticos sobre como usar GitHub Desktop, a ferramenta visual para gerenciar repositórios Git.

   Objetivo

Fornecer um guia prático e acessível para iniciantes aprenderem a usar GitHub Desktop de forma eficiente.

   Conteúdo

- Instalação e configuração
- Conceitos básicos de Git
- Clonagem de repositórios
- Criação de branches
- Commits e push
- Pull requests
- Resolução de conflitos
- Boas práticas

    Pré-requisitos

- GitHub Desktop instalado
- Conta GitHub
- Conhecimento básico de versionamento

    Como Usar

1. Clone este repositório
2. Abra os arquivos de tutorial
3. Siga os exemplos práticos
4. Pratique os conceitos

    Tópicos Cobertos

      Iniciante
- O que é Git e GitHub
- Instalação do GitHub Desktop
- Configuração inicial
- Primeiro repositório

     Intermediário
- Branches e merges
- Commits efetivos
- Push e pull
- Sincronização

     Avançado
- Resolução de conflitos
- Rebase e cherry-pick
- Stash
- Histórico e blame

    Dicas Úteis

- Use mensagens de commit descritivas
- Faça commits pequenos e frequentes
- Sempre puxe antes de fazer push
- Revise mudanças antes de commitar
